# Java Text Editor

This project is a simple text editor developed in Java using the Swing framework. It was created as an academic project to practice graphical user interface (GUI) development and file manipulation using Java I/O.

The application allows users to open, edit, and save text files through a simple and intuitive interface.

## Features

- Open text files
- Edit file contents
- Save text files
- File selection using `JFileChooser`
- Simple graphical interface built with Java Swing

## Technologies

- Java
- Java Swing
- Java I/O
- NetBeans

## Learning Objectives

The project was developed to practice:

- Graphical user interface development with Swing
- Reading and writing files using Java I/O
- Event-driven programming
- Desktop application development in Java

## Getting Started

1. Clone this repository:

```bash
git clone https://github.com/your-username/java-text-editor.git
```

2. Open the project in NetBeans or any Java IDE.

3. Build and run the application.
